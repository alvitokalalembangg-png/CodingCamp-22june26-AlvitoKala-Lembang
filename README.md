# ExpenseViz — Expense & Budget Visualizer
A mobile-friendly expense tracker designed to help users manage daily spending with visual charts and budget alerts.

## Live Demo
🔗 https://alvitokalalembangg-png.github.io/CodingCamp-22june26-AlvitoKala-Lembang/

## Features
- Add and delete transactions by category
- Real-time pie chart visualization (Chart.js)
- Spending limit tracker with alerts
- Monthly spending summary
- Custom category creation
- Sort transactions by newest, oldest, amount, or category
- Dark and light theme toggle
- Data saved with Local Storage
- Responsive user interface

## Technologies Used
- HTML5
- CSS3
- JavaScript
